Theralinq

A mobile app for synchronized task-tracking and communication between therapists and caregivers.

Live Link

Project Overview

The goal of this project was to develop an MVP of a mobile app for therapists to communicate with caregivers of their patients, with task-tracking capabilities to facilitate continued engagement with therapy exercises beyond in-person therapy sessions.

The scope of this project included customer interviews, UI system design, low-fi/high-fi wireframing, backend database design, creating the app using a low-code platform, writing tech specs, and internal and external user testing on TestFlight.
